# Correlation Reports

Daily threat reports generated by cross-sensor correlation across all honeypot sensors:
Cowrie · Dionaea · OpenCanary · Telnethoney · Suricata.

Each report identifies IPs that appeared on multiple sensors within the same day,
classifies transition timing (real-time vs delayed), and summarises attacker behaviour.

## Sample report

- [daily_2026-04-11_anon.md](daily_2026-04-11_anon.md) — rendered inline on GitHub
- [daily_2026-04-11_anon.html](daily_2026-04-11_anon.html) — full dark-theme dashboard (open locally)

### What the report shows

| Field | Description |
|-------|-------------|
| Risk level | HIGH / MEDIUM / LOW based on transition count and timing |
| Transitions | Which sensors the IP moved between and in what order |
| avg_delay | Average time between sensor hits in seconds |
| type | REAL_TIME (<1s) or LONG_DELAY (>60s) |
| interpretation | Automated bot vs unrelated events |

### Example finding

An IP hitting Dionaea → Cowrie → Telnethoney within milliseconds indicates
a single automated process scanning multiple protocols simultaneously —
not three separate attackers sharing the same address.

## Anonymisation

All source IPs replaced with 203.0.113.x (RFC 5737 documentation range).
Report date and structure preserved as-is.
